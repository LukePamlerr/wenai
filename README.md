# Overtime - Roblox Studio Website

A premium, high-fidelity website for Overtime, a Roblox development studio specializing in partnerships, LiveOps, game development, and acquisitions.

## Features

- **Dark Theme**: Sleek black and white aesthetic.
- **Live Stats**: Real-time player counts and visit data from Roblox games.
- **Portfolio**: Showcasing industry-leading experiences.
- **Team**: Meet the elite collective behind the studio.
- **Applications**: Custom developer application flow.

## Deployment

### Deploy to Vercel

You can deploy this website to Vercel with one click:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FLukePamlerr%2Fwenai)

> **Note**: To make the "Deploy to Vercel" button work with your own repository, replace the `repository-url` in the link above with your GitHub repository URL.

### Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Motion.
- **Backend**: Express (for API proxying).
- **Icons**: Lucide React.
